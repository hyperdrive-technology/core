# Inrush Documentation

Welcome to the Inrush documentation. Inrush is a modern, open-source PLC runtime with support for IEC 61131-3 programming languages and online changes.

## Features

- IEC 61131-3 programming languages support
- Real-time control and monitoring
- Online program changes (similar to Logix)
- WebSocket-based communication
- Modern React-based UI

## Getting Started

To get started with Inrush, follow these steps:

1. Install the prerequisites:
   - Node.js 18+
   - Go 1.21+
   - Docker & Docker Compose

2. Clone the repository:
   ```bash
   git clone https://github.com/inrush-io/inrush.git
   cd inrush
   ```

3. Install dependencies:
   ```bash
   pnpm install
   ```

4. Start the development environment:
   ```bash
   docker compose up
   ```
