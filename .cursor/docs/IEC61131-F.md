### Caption

Table F.2. Other implementation-dependent parameters (not part of IEC 61131-3). (Continued on next page)

### Introduction

There are a large number of other parameters that need to be considered when implementing a programming system compliant with IEC 61131-3.
Table F.2 gives a subjective selection of these parameters.

### Data

| No.                        | Implementation-Dependent Parameters                                                                                                                                                                                                                                                                                                                                                  |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| <pre><code>1</code></pre>  | Extent of syntax and semantic checking provided by the textual and graphical editors (during input).                                                                                                                                                                                                                                                                                 |
| <pre><code>2</code></pre>  | Description of the execution order of networks.                                                                                                                                                                                                                                                                                                                                      |
| <pre><code>3</code></pre>  | Free placement of graphic elements in the case of editors with line updating ("elastic band") or automatic placement according to syntax rules.                                                                                                                                                                                                                                      |
| <pre><code>4</code></pre>  | Declaration and use of Directly Represented Variables (DRVs):<br>- In the declaration the symbolic name is always used. Or<br>- The programming system declares DRVs implicitly. |
